# Triple_Backend Homework 2

Flask + MySQL

## 실행 전 아래명령어를 사용하여 필요 패키지들을 설치해주세요.

$ pip3 install -r requirements.txt

아래 명령어로 실행가능합니다.
$ python3 main.py
