# Triple_Backend Homework 2

사용언어 : Python3.9
Flask + MySQL

## 실행 전 아래명령어를 사용하여 필요 패키지들을 설치해주세요.

$ pip3 install -r requirements.txt

아래 명령어로 실행가능합니다.
$ python3 main.py

첫 실행시 테이블 생성에 시간이 걸릴 수 있습니다.
생성테이블은 review, user, review_photo, review_log 4개입니다.
